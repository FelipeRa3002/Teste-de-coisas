Projeto Login

Curso em Vídeo

Projeto construído para treinar o uso de iframes e interfaces interativas.

Saiba mais sobre o Projeto Login
Sobre o Projeto Login

Este site foi desenvolvido como um desafio prático de encerramento do Módulo 4 do curso de HTML5 e CSS3 do canal Curso em Vídeo, sob a orientação didática do professor Gustavo Guanabara.

O que foi aplicado:
  • Uso de Iframes: Domínio da tag <iframe> para carregar páginas externas e locais dinamicamente, permitindo a navegação interna sem sair da interface principal.
  • Simulação de Hardware (Smartphone): Criação de uma moldura de celular realista utilizando propriedades avançadas de CSS, como border-radius, box-shadow e posicionamento relativo/absoluto para as imagens de fundo.
  • Design Responsivo Avançado: Aplicação de Media Queries para garantir que o "celular virtual" se ajuste proporcionalmente ou seja ocultado em telas muito pequenas, mantendo a usabilidade.
  • Scroll Customizado: Técnicas de estilização de barras de rolagem para que a navegação dentro do iframe pareça um aplicativo nativo.
  • Interatividade com Alvos (Target): Uso do atributo target em links para manipular qual conteúdo deve ser exibido dentro do iframe nomeado.

O foco principal foi aprender a lidar com elementos embutidos e entender como o CSS pode transformar estruturas simples em interfaces que simulam objetos do mundo real (skeuomorfismo).

Você pode conferir o código fonte completo e a versão ao vivo no meu GitHub.

Menu FlexBox

Curso em Vídeo

Projeto desenvolvido para dominar layouts modernos e flexíveis utilizando a tecnologia CSS Flexbox.

Saiba mais sobre o Projeto Menu Flexbox
Sobre o Projeto Menu Flexbox

Este projeto foi construído durante o Módulo 5 do curso de HTML5 e CSS3 do Curso em Vídeo, focando na transição do design estático para o design flexível.

O que foi aplicado:
  • Containers Flexíveis: Implementação do display: flex para controle total sobre o alinhamento de itens de menu.
  • Alinhamento Dinâmico: Uso de justify-content: space-between para distribuir o logo e a navegação de forma automática e elegante.
  • Responsividade Fluida: Adaptação do menu para o modo "coluna" (column) em telas menores sem a necessidade de floats ou hacks antigos.
  • Hierarquia Visual: Organização de elementos de cabeçalho utilizando as propriedades de alinhamento vertical align-items: center.

O objetivo deste projeto foi abandonar técnicas obsoletas de layout e adotar o padrão moderno da indústria para componentes de navegação.
